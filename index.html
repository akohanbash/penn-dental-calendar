What it does

A lightweight Chrome extension that turns your Penn Dental schedule page into a single, importable calendar file — styled the way you want it.

🗓️

One-file export

Reads your signed-in schedule and generates a single .ics file, ready to import into any calendar app.

🎨

Tap-to-pick emoji icons

Open the emoji editor page and tap any event type to choose its icon from a picker grid — no typing.

✨

Five preset options

Classic, Academic, Minimal, Dental, or No Emoji — turn icons off entirely and use Penn's original titles.

🗂️

Dedicated editor page

Emoji customization lives on its own screen, keeping the main export screen simple and fast.

🌗

Light & dark interface

An Apple-inspired popup that automatically matches your system appearance.

🔁

Reset anytime

Reset a single emoji or your entire configuration back to defaults in one click.

Install in under a minute

Chrome extensions loaded this way aren't published on the Web Store, so installation is done manually through Chrome's Developer Mode. It only takes a few clicks.

Download the extension

Click Code → Download ZIP on the GitHub repository, then unzip the file on your computer.

Open Chrome's extensions page

Go to chrome://extensions in the address bar.

Turn on Developer mode

Flip the Developer mode switch in the top-right corner of the page.

Load the extension

Click Load unpacked, then select the extension folder from the files you unzipped.

Export your schedule

Open your Penn Dental schedule page, click the extension icon, customize your emojis if you'd like, and download your calendar file.

Privacy first

Nothing about your schedule ever leaves your browser.

🚫

No servers

All processing happens locally on your device.

🔑

No password required

The extension only reads the page you're already signed into.

💾

Preferences stay local

Your emoji settings are saved in your own browser, not in the cloud.
